## 🚀 Featured Projects

### 🤖 [Mattermost OWUI Chatbot](https://github.com/ashutoshmaske22/Mattermost-OWUI-chatbot)

> Connect any Open WebUI model to Mattermost as a real chatbot. DMs, thread following, and conversation memory. No plugin. No enterprise license. Pure Python.

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)
![WebSocket](https://img.shields.io/badge/WebSocket-000000?style=flat)
![Open WebUI](https://img.shields.io/badge/Open%20WebUI-000000?style=flat&logo=openai&logoColor=white)
![Mattermost](https://img.shields.io/badge/Mattermost-0058CC?style=flat&logo=mattermost&logoColor=white)
![MIT License](https://img.shields.io/badge/License-MIT-green?style=flat)

Our team ran Open WebUI with Llama 3.2 locally. Nobody used it — 4 clicks to reach it every time. So I connected it directly into Mattermost via WebSocket + REST API. No plugin required, no enterprise plan.

- DM the bot → private conversation with full memory + `/reset` to clear
- `@mention` once in a thread → bot follows the whole thread automatically, no trigger words needed after that
- Full conversation history sent to the model on every reply — it always knows the context
- Works with any model in Open WebUI — Llama, Mistral, Gemma, DeepSeek
- 1 Python file · 1 dependency · everything configured via `.env`
