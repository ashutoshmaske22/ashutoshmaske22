# Contributing to DevOps Mastery Hub

Thanks for checking this out! This is primarily a personal learning and reference repo, but PRs and suggestions are welcome.

## 📁 Structure Convention

Each topic folder should have:
```
topic-name/
├── README.md         # notes + cheatsheet + interview Q&A
├── scripts/          # runnable scripts or code
└── examples/         # project configs, manifests, pipelines
```

## ✍️ Content Guidelines

- Keep notes **intermediate level** — assume reader knows basics
- Every section should have **runnable examples** where possible
- End every topic README with **5 interview questions** (with answers)
- Use code comments to explain non-obvious parts

## 🐛 Found a bug or error?

Open an issue or a PR with the fix — much appreciated!
